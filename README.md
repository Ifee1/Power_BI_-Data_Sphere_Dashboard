Restaurant Inspection Analysis Dashboard (Power BI)

Project Overview

This project analyzes restaurant inspection data using Power BI to explore inspection activity, violation patterns, sanitation performance, and regional compliance trends.
The dashboard provides a clear view of how inspections are distributed across regions, the most common types of violations, and overall sanitation performance across inspected establishments.
________________________________________
Business Problem

Health authorities conduct inspections to ensure restaurants comply with food safety regulations. However, large volumes of inspection data can make it difficult to quickly identify:
•	Regions with the highest inspection activity
•	The most common types of violations
•	Sanitation performance across establishments
•	Whether inspection targets are being met
This dashboard helps transform inspection data into clear visual insights that support monitoring and regulatory decision-making.

Key Analytical Focus Areas

The dashboard explores several important operational questions:
•	Which regions receive the highest number of inspections?
•	Which violation categories occur most frequently?
•	How are sanitation grades distributed across establishments?
•	Are inspection targets being met?
•	How do inspection trends change across regions?
________________________________________

Key Insights
Some key insights derived from the dashboard include:
•	Los Angeles records the highest inspection activity (~13.8K inspections).
•	The South Bay region exceeds its inspection target by ~17%.
•	Facility condition violations are among the most common issues observed.
•	Grade A sanitation ratings dominate, indicating strong compliance overall.
•	The average sanitation score (92.21) is slightly above the target (91.91).
